## Criação de tabela

MySQL

```
CREATE TABLE users (
    id INT AUTO_INCREMENT PRIMARY KEY,         
    name VARCHAR(255) NOT NULL,                 
    email VARCHAR(255) NOT NULL UNIQUE,         
    birth_date DATE NOT NULL,                   
    age INT CHECK (age >= 0),                   
    has_driver_license TINYINT(1) DEFAULT 0
);
```

    email VARCHAR(255) NOT NULL UNIQUE,
    birth_date DATE NOT NULL,
    age INT CHECK (age >= 0),
    has_driver_license BOOLEAN DEFAULT FALSE
